## 조별 회고

### Keep
- 기능 추가
  - 구현해보지 않았던 기능들을 프로젝트를 진행하며 추가하는 과정에서 많은 성장이 있었던 것 같다.
- 커뮤니케이션 (존중과 배려를 기반)
  - 활발한 의사소통으로 문제 없이 진행이 되었습니다.
  - 백, 프론트 구분 없이 이슈가 생기면 함께 고민하고 이슈 해결
- 일단 해보자! 하고 싶은건 하자!
  - 일단 해보자 하는 건 다 구현해낸 집요함. 안되면 빠르게 수정.
  

### Problem
- 모델 설정
  - 모델을 미리 대부분 설정을 해놓고 진행을 했어야 했지만, 중간중간 기능추가가 들어가면서 모델 수정이 너무 빈번했다.
- 일정
  - 계획대로 진행했지만 중간중간 사소한 오류들이 발생하여 일정이 밀렸음

### Try
- 타겟 설정과 모델 설정
  - 프로젝트 진행 전 타겟 설정을 미리 하고, 필요한 기능이 무엇인지 정리한다.
  - 필요한 기능에 필요한 필드를 생각하고 모델 설정에 시간을 들인다.
- 사용자 편의성과 사이트의 유저 관심도
  - 사용자가 기능을 이용하기 쉽도록 편의성을 고려하여 배치
  - 유저 관심도를 높여 많은 사람들이 가입하여 사이트를 사용하도록 설정
  - 반응형 웹페이지 구현
- 소셜 로그인
  - 통합 로그인 만들기
  - 마지막에 어떤 소셜로 로그인 되었는지 표시
- 채팅 구현
  - 로컬 환경이 아닌 배포 사이트에서도 채팅 구현

## 개인별 회고

### 고해림 (Backend)

> 먼저, 프로젝트 시작하기 전 많이 부족했던 제가 좋은 팀원분들을 만나 한 단계 성장할 수 있었습니다. 로그인, 지도 api, 무적의 프론트엔드, 채팅 등 저희 조 아마 이번 프로젝트 중 조별 총 수면 시간 꼴등 예상, 아니 확신합니다. 해시태그(tag) 구현 시 model 관련 이슈가 있었는데 팀원분들과 함께라면 뭐든지 해결 가능했습니다. 이번 프로젝트를 통해 팀 프로젝트가 이렇게 재밌고 고마운 일인지 또 다시 깨닫게 되었습니다. 매일 밤 저 대신 2조에 머물러주신 명예 2조분들께도 감사의 인사를 전합니다. 다 된 밥상에 밥 너무 맛나게 먹었습니다 …

### 문지수 (Backend)

> 이번 프로젝트를 진행하며 너무 좋은 팀원들을 만나 팀장을 맡아도 별로 할 일이 없었습니다. 제가 말씀을 드리기도 전에 이미 할 일을 찾아서 하시고, 의욕적으로 같이 새벽까지 프로젝트를 진행하며 매우 좋았던 경험을 했다고 생각합니다. 프로젝트 중간 중간 많은 문제가 발생했지만, 팀원 분들과 같이 소통하며 문제를 해결할 수 있었고, 협업의 장점을 깨닫게 되었습니다. 일주일이라는 짧은 시간이었지만 필터링, URL 쿼링, 검색, 메시지 등을 구현하며 조금의 성장을 하게 된 것 같습니다. 앞으로의 프로젝트가 조금 더 기대되는 경험이었습니다.

### 장민지 (Frontend)

> 프로젝트 하기 전엔 두려운 마음만 있었는데 막상 해보니 지금은 뿌듯한 마음만 있습니다! 좋은 팀원 분들 덕에 정말 많은 걸 배웠고! 이런 팀을 또 만날 수 있을까 할 정도로 정말 다 완벽 했습니다! 다들 서로 배려하면서 프로젝트를 진행하여 좋은 분위기 속에 좋은 결과를 얻을 수 있었습니다! 실력 있는 팀원들에 비해 많이 부족한 실력이지만 어느 누구도 탓하지 않고 잘한 것만 봐주는 팀원 분들 덕에 정말 재밌는 프로젝트였습니다. 프로젝트에서 구현 된 채팅, 쪽지, 지도api, 프론트엔드 모두 앞으로 저의 배움에 좋은 방향성이 되어 줄 것 같습니다! 프로젝트가 끝나서 다시 한번 더 복습을 해야 할 정도로 정말 많은 게 구현된 홈페이지를 함께 만들 수 있어서 영광이었습니다! 감사해요 모두들!!

### 정소담 (Frontend)

> 이번 프로젝트 동안 대부분의 시간을 팀원들과 함께 보낸 것 같습니다. 팀 프로젝트의 장점만 느꼈던 시간이었어요. 채팅이나 별점, 이모지 등을 구현 할 때 문제가 발생해도 포기하지 않고 끝까지 다같이 해결했던 게 기억에 많이 남습니다.  능력 있는 팀원들을 만난 덕분에 짧은 시간에도 다양한 기능을 접해볼 수 있었습니다. 다음에는 반응형 웹, 쿼리 최적화 등 더 챙겨보고 싶은 것들도 많아 졌습니다. 많은 것을 얻어갑니다. 여러분 사랑해요 …♥
